1) File name convention :  
   I3RC_cons_res_case2_exp#_RQ.txt 
   i)   "exp#" is the experiment number as listed above; valid number are 1-8 for this case
   ii)  "RQ" is the radiative quantity contained the file. RQ takes the following values:
         RQ=R     (reflectance)
         RQ=T     (transmittance)
         RQ=A     (absorptance)
         RQ=H     (net horizontal flux)
         RQ=Iu    (nadir reflectivity)
         RQ=Id    (zenith transmissivity)  

2) The file formats are described at http://i3rc.gsfc.nasa.gov/input/MMCR/high_res/020898/README.txt
