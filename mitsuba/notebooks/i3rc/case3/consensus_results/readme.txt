1) File name convention :  
   I3RC_cons_res_case3_exp#_RQ.txt 
   i)   "exp#" is the experiment number as listed above; valid number are 1-4 for this case
   ii)  "RQ" is the radiative quantity contained the file. RQ takes the following values:
         RQ=R     (reflectance)
         RQ=T     (transmittance)
         RQ=A     (absorptance)
         RQ=H     (net horizontal flux)
         RQ=Iu    (nadir reflectivity)
         RQ=Id    (zenith transmissivity)  

2) The file formats are described at http://i3rc.gsfc.nasa.gov/input/Landsat/README.txt

3) This distribution includes the following consensus results: 
   I3RC_cons_res_case3_exp1_Iu.txt
   I3RC_cons_res_case3_exp1_R.txt
   I3RC_cons_res_case3_exp1_T.txt
   I3RC_cons_res_case3_exp2_Id.txt
   I3RC_cons_res_case3_exp2_Iu.txt
   I3RC_cons_res_case3_exp3_Iu.txt
   I3RC_cons_res_case3_exp3_R.txt
   I3RC_cons_res_case3_exp3_T.txt  
   I3RC_cons_res_case3_exp4_A.txt
   I3RC_cons_res_case3_exp4_R.txt
   I3RC_cons_res_case3_exp4_T.txt

4) For comparisons and/or I3RC certificates of accuracy for Case 3 results not included in this distribution, 
   please contact Stefani Huang (stefani_huang@ssaihq.com) or Tamas Varnai (tamas.varnai@nasa.gov).
